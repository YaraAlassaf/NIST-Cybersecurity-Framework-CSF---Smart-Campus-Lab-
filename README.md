# NIST Cybersecurity Framework (CSF) – Smart Campus Lab

## Project Overview

This project demonstrates the application of the NIST Cybersecurity Framework (CSF) to assess and improve the cybersecurity posture of a Smart Campus laboratory environment. The assessment covers the five core NIST CSF functions by analyzing the current security posture, defining a target profile, and recommending practical actions to enhance the organization's cybersecurity resilience.

## Scope

The Smart Campus environment includes:

- University laboratory infrastructure
- Student applications
- IoT devices
- Bring Your Own Device (BYOD)
- Cloud-based services

## NIST CSF Functions

- Identify – Asset Management
- Protect – Access Control
- Detect – Security Continuous Monitoring
- Respond – Response Planning
- Recover – Recovery Planning

## Key Recommendations

- Implement automated asset inventory and device discovery.
- Enforce Multi-Factor Authentication (MFA) and Role-Based Access Control (RBAC).
- Enable continuous network monitoring with automated alerts.
- Develop and maintain a formal Incident Response Plan.
- Improve backup and recovery processes, including Recovery Time Objectives (RTO) and post-incident reviews.
